Utilizzo dei tipi di dati (JDBC)

La funzione principale di Microsoft JDBC Driver per SQL Server è consentire agli sviluppatori Java di accedere a dati contenuti in database di SQL Server. A tale scopo, il driver JDBC consente di eseguire la conversione tra i tipi di dati di SQL Server e i tipi e gli oggetti del linguaggio Java.

Per informazioni dettagliate sui tipi di dati SQL Server e sui tipi di dati del driver JDBC, incluse le relative differenze e in che modo vengono convertiti in tipi di dati del linguaggio Java, vedere Informazioni sui tipi di dati del driver JDBC.

Per poter utilizzare i tipi di dati di SQL Server, il driver JDBC fornisce i metodi get<Type> e set<Type> per le classi SQLServerPreparedStatement e SQLServerCallableStatement e i metodi get<Type> e update<Type> per la classe SQLServerResultSet. Il metodo da adottare dipende dal tipo di dati che si sta utilizzando e dall'eventuale utilizzo di set di risultati o query.

Negli argomenti di questa sezione viene descritto come utilizzare i tipi di dati del driver JDBC per accedere ai dati SQL Server nelle applicazioni Java.
