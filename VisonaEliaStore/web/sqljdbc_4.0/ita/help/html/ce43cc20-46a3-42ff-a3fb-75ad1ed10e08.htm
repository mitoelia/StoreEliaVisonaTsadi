Connessione e recupero dei dati

Quando si utilizza Microsoft JDBC Driver per SQL Server, sono disponibili due metodi principali per stabilire una connessione a un database di SQL Server. Il primo metodo consente di impostare le proprietà di connessione nell'URL della connessione, quindi di eseguire una chiamata al metodo getConnection della classe DriverManager per restituire un oggetto SQLServerConnection.

Per un elenco delle proprietà di connessione supportate dal driver JDBC, vedere Impostazione delle proprietà delle connessioni.

Il secondo metodo consente di impostare le proprietà di connessione mediante i metodi per l'impostazione della classe SQLServerDataSource, quindi di eseguire la chiamata al metodo getConnection per restituire un oggetto SQLServerConnection.

Negli argomenti di questa sezione vengono descritte le diverse modalità di connessione a un database di SQL Server e vengono inoltre illustrate le diverse tecniche per il recupero dei dati.
