Classe SQLServerResultSet

Rappresenta un set di risultati JDBC.

Pacchetto: com.microsoft.sqlserver.jdbc

Implementa:ISQLServerResultSet

Sintassi

public final class SQLServerResultSet

Osservazioni

Esistono due tipi di set di risultati: lato client e lato server.

I set di risultati del lato client vengono utilizzati quando le dimensioni dei risultati sono tali da consentirne la conservazione nella memoria del processo client. Questi risultati sono disponibili più rapidamente e vengono letti interamente dal database da Microsoft JDBC Driver per SQL Server. Inoltre non impongono carico aggiuntivo sul database, evitando l'overhead correlato alla creazione di cursori sul lato server. Tuttavia, questi tipi di set di risultati non sono aggiornabili.

I set di risultati del lato server possono essere utilizzati quando le dimensioni dei risultati sono tali da non consentirne la conservazione nella memoria del processo client o quando il set di risultati deve essere aggiornabile. Con questo tipo di set di risultati, il driver JDBC crea un cursore sul lato server e recupera in modo trasparente le righe del set di risultati man mano che l'utente scorre il set stesso.

La classe SQLServerResultSet fornisce numerosi metodi che consentono l'aggiornamento del set di risultati con qualsiasi tipo di dati Java nativo e numerosi tipi di oggetti Java.

Questa classe supporta l'annullamento del wrapping nella classe SQLServerResultSet e nelle interfacce ISQLServerResultSet e java.sql.ResultSet. Per ulteriori informazioni, vedere Wrapper e interfacce.
