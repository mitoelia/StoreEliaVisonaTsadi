Metodo getConnection (java.lang.String, java.lang.String)

Tenta di stabilire una connessione con l'origine dati rappresentata da questo oggetto SQLServerDataSource utilizzando il nome utente e la password specificati.

Sintassi

public java.sql.Connection getConnection(java.lang.String username,
                                         java.lang.String password)

Parametri

username

Valore String contenente il nome utente.

password

Valore String contenente la password.

Valore restituito

Oggetto SQLServerConnection.

Eccezioni

java.sql.SQLException

Osservazioni

Questo metodo getConnection viene specificato dal metodo getConnection nell'interfaccia javax.sql.DataSource.

La chiamata al metodo getConnection con una password o nome utente non Null sostituirà le proprietà del nome utente e della password che vengono impostate sulla classe SQLServerDataSource quando si inizializza l'oggetto SQLServerConnection. Ad esempio, se il chiamante ha chiamato setUser e setPassword sull'origine dati e successivamente chiama getConnection e fornisce un nome utente non Null o una password non Null, il nome utente e la password impostati da setUser e setPassword saranno sostituiti dal nome utente e password passati in getConnection.

Il nome utente e la password che sono impostati nell'URL tramite una chiamata al metodo setURL non saranno modificati in questo caso.
