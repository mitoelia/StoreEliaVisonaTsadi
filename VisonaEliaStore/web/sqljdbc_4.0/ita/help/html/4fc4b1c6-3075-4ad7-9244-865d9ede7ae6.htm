Utilizzo dei set di risultati

Quando si utilizzano i dati contenuti in un database di SQL Server, un metodo di manipolazione dei dati consiste nell'utilizzo di un set di risultati. Microsoft JDBC Driver per SQL Server supporta l'utilizzo dei set di risultati tramite l'oggetto SQLServerResultSet. Utilizzando l'oggetto SQLServerResultSet, è possibile recuperare i dati restituiti da un'istruzione SQL o da una stored procedure, aggiornare i dati secondo le necessità, quindi inviare nuovamente i dati al database. 

Inoltre, l'oggetto SQLServerResultSet fornisce metodi per la navigazione nelle righe di dati, per il recupero o l'impostazione dei dati in esso contenuti, per la creazione di vari livelli di sensibilità alle modifiche nel database sottostante.

Per ulteriori informazioni sulla gestione dei set di risultati, inclusa la sensibilità alle modifiche, vedere Gestione dei set di risultati con il driver JDBC.

Negli argomenti della sezione vengono descritti vari modi di utilizzare un set di risultati per manipolare i dati contenuti in un database SQL Server.
