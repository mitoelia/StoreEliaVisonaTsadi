Panoramica del driver JDBC

Microsoft JDBC Driver per SQL Server è un driver conforme a Java Database Connectivity (JDBC) 4.0 che offre potenti funzionalità di accesso ai dati per i database di Microsoft SQL Server. Il driver JDBC consente di utilizzare molte delle caratteristiche introdotte in SQL Server 2005, inclusi il mirroring del database, i tipi di dati XML, definiti dall'utente e per valori di grandi dimensioni. Supporta inoltre il nuovo tipo di isolamento delle transazioni "snapshot". Inoltre, il driver JDBC supporta anche l'utilizzo dell'autenticazione integrata.

Negli argomenti di questa sezione viene fornita una panoramica generale del driver JDBC. Sono inclusi i requisiti di sistema necessari, le modalità di utilizzo del driver e la reperibilità di ulteriori informazioni.
