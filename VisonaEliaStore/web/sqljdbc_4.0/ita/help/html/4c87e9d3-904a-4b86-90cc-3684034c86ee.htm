Classe SQLServerPooledConnection

Rappresenta una connessione di database fisica in un pool di connessioni.

Pacchetto: com.microsoft.sqlserver.jdbc

Estende: java.lang.Object

Implementa: javax.sql.PooledConnection

Sintassi

public final class SQLServerPoolingDataSource

Osservazioni

La classe SQLServerPooledConnection fornisce metodi affinché il pool di connessioni venga gestito dall'apposito strumento. In genere le applicazioni non creano direttamente un'istanza di queste connessioni.
