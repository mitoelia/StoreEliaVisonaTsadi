Metodo setObject (java.lang.String, java.lang.Object, int, int)

Imposta il valore del parametro designato tramite l'oggetto, il tipo di destinazione e la scala specificati.

Sintassi

public void setObject(java.lang.String sCol,
                      java.lang.Object o,
                      int n,
                      int m)

Parametri

sCol

Valore String contenente il nome del parametro.

o

Valore Object .

n

Valore int che indica il tipo di destinazione come definito in java.sql.Types.

m

Valore int che indica il numero di cifre a destra del separatore decimale. Questo parametro viene ignorato per tutti i tipi a eccezione di NUMERIC e DECIMAL.

Eccezioni

SQLServerException

Osservazioni

Questo metodo setObject viene specificato dal metodo setObject nell'interfaccia java.sql.CallableStatement.

A partire dal driver JDBC 3.0 per SQL Server, il comportamento di questo metodo viene modificato dalla proprietà di connessione sendTimeAsDatetime (Impostazione delle proprietà delle connessioni) e dall'oggetto SQLServerDataSource.setSendTimeAsDatetime.

Per ulteriori informazioni, vedere Configurazione della modalità di invio dei valori java.sql.Time al server.
