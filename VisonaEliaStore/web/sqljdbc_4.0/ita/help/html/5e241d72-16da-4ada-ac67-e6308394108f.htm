Recupero della versione del driver

È possibile individuare la versione installata di Microsoft JDBC Driver per SQL Server nei modi seguenti:

  • Chiamando i metodi SQLServerDatabaseMetaDatagetDriverMajorVersion, getDriverMinorVersion o getDriverVersion.

  • La versione viene visualizzata nel file readme.txt della distribuzione del prodotto.

Il nome del driver JDBC può inoltre essere restituito dalla chiamata del metodo getDriverName nella classe SQLServerDatabaseMetaData. La chiamata restituirà "Microsoft JDBC Driver 4.0 for SQL Server".

Di seguito è riportato un esempio di output delle chiamate ai metodi della classe SQLServerDatabaseMetaData:

getDriverName = Microsoft JDBC Driver 4.0 for SQL Server

getDriverMajorVersion = 4

getDriverMinorVersion = 0

getDriverVersion = 4.0.xxx.x

Dove "xxx.x" è il numero della versione finale.
