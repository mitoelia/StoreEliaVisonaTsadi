Metodo setPortNumber (SQLServerDataSource)

Imposta il numero di porta da utilizzare per comunicare con SQL Server.

Sintassi

public void setPortNumber(int portNumber)

Parametri

portNumber

Valore int contenente il numero di porta.

Osservazioni

Il numero di porta è il numero di porta TCP/IP utilizzato quando si apre una connessione socket a SQL Server. Se la proprietà portNumber non è impostata, il metodo getPortNumber restituisce il valore predefinito 1433.

Il metodo setPortNumber non esegue alcun controllo di intervallo sul valore di porta passato. È possibile passare un numero di porta non valido, come 99999, senza generare un errore.
