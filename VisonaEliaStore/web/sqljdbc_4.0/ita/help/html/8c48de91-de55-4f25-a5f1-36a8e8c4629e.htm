Metodo getReference (SQLServerConnectionPoolDataSource)

Restituisce un riferimento a questo oggetto SQLServerConnectionPoolDataSource.

Sintassi

public javax.naming.Reference getReference()

Valore restituito

Oggetto Reference.

Osservazioni

Questo metodo getReference viene specificato dal metodo getReference nell'interfaccia javax.naming.Referenceable. Esegue l'override del metodo getReference della classe SQLServerDataSource.
