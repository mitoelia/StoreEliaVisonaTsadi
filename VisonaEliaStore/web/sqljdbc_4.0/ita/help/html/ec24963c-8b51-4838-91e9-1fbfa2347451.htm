Classe SQLServerStatement

Rappresenta l'implementazione di base della funzionalità dell'istruzione JDBC.

Pacchetto: com.microsoft.sqlserver.jdbc

Implementa:ISQLServerStatement

Sintassi

public class SQLServerStatement

Osservazioni

La classe SQLServerStatement fornisce anche alcuni metodi di implementazione della classe di base per l'istruzione preparata di JDBC e per le istruzioni di JDBC disponibili per la chiamata. Il ruolo di base della classe SQLServerStatement prevede l'esecuzione di istruzioni SQL, quindi la restituzione dei conteggi aggiornamenti e dei set di risultati all'applicazione utente.

Questa classe supporta l'annullamento del wrapping nella classe SQLServerStatement e nelle interfacce ISQLServerStatement e java.sql.Statement. Per ulteriori informazioni, vedere Wrapper e interfacce.
