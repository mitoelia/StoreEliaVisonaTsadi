Utilizzo di istruzioni SQL

Quando si gestiscono dati in un database di SQL Server tramite Microsoft JDBC Driver per SQL Server e istruzioni SQL inline, è possibile utilizzare diverse classi. in base al tipo di istruzione SQL che si desidera eseguire.

Se l'istruzione SQL non contiene parametri IN, utilizzare la classe SQLServerStatement; se, invece, contiene parametri IN, utilizzare la classe SQLServerPreparedStatement.

Se è necessario utilizzare istruzioni SQL che contengono sia parametri IN che OUT, è necessario implementarle come stored procedure e chiamarle utilizzando la classe SQLServerCallableStatement. Per ulteriori informazioni sull'utilizzo di stored procedure, vedere Utilizzo delle istruzioni con le stored procedure.
