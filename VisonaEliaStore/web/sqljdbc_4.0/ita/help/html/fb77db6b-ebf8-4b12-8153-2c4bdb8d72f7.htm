Costruttore SQLServerSavepoint (SQLServerConnection, java.lang.StringName)

Sintassi

public SQLServerSavepoint(SQLServerConnection con,
                          java.lang.String sName)

Parametri

con

Oggetto SQLServerConnection.

sName

Valore String che contiene il nome del punto di salvataggio.
