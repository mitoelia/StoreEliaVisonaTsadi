Metodo isWrapperFor (SQLServerCallableStatement)

Indica se questo oggetto istruzione è un wrapper per l'interfaccia specificata.

Sintassi

public boolean isWrapperFor(Class iface)

Parametri

iface

Valore class che definisce un'interfaccia.

Valore restituito

true se l'oggetto implementa l'interfaccia o esegue il wrapping di un oggetto che implementa l'interfaccia. In caso contrario, false.

Eccezioni

SQLServerException

Osservazioni

Il metodo isWrapperFor e il metodo unwrap sono definiti dall'interfaccia java.sql.Wrapper, introdotta in JDBC 4.0.

Se questo metodo restituisce true, la chiamata a unwrap con lo stesso argomento verrà eseguita correttamente.

Per ulteriori informazioni, vedere Wrapper e interfacce.
