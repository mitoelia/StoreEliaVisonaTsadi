Metodo setResponseBuffering (ISQLServerStatement)

Imposta la modalità di memorizzazione delle risposte nel buffer per questo oggetto SQLServerStatement su String full o adaptive senza distinzione tra maiuscole e minuscole.

Sintassi

public final void setResponseBuffering(java.lang.String value)

Parametri

value

Valore String contenente la modalità di memorizzazione delle risposte nel buffer. La modalità valida può essere una delle stringhe senza distinzione tra maiuscole e minuscole seguenti: full o adaptive.

Eccezioni

SQLServerException

Osservazioni

adaptive specifica la memorizzazione nel buffer della quantità di dati minima possibile, quando necessario.

full specifica la lettura dell'intero risultato dal server in fase di esecuzione.

adaptive è il valore predefinito nelle versioni 2.0 e 3.0 del driver JDBC, mentre full è il valore predefinito nelle versioni del driver JDBC antecedenti a 2.0.

Il metodo setResponseBuffering consente di eseguire l'override della proprietà String della connessione responseBuffering per l'oggetto SQLServerStatement corrente. Per ulteriori informazioni sull'utilizzo della modalità di memorizzazione nel buffer, vedere Utilizzo del buffer adattivo.

Se l'applicazione specifica un valore del parametro non valido nel metodo setResponseBuffering, viene generato un oggetto SQLServerException.
