Campo SSTRANSTIGHTLYCPLD (SQLServerXAResource)

Utilizzato per consentire le transazioni XA "a regime di controllo stretto ("tightly-coupled")" che dispongono di diversi ID di transazione dei rami XA (XID), ma dello stesso ID di transazione globale (GTRID).

Sintassi

public static final int SSTRANSTIGHTLYCPLD

Valore del campo

Valore int pari a 32768.

Osservazioni

Ogni transazione viene identificata da un ID di transazione dei rami XA (XID) e da un ID di transazione globale (GTRID). Per consentire alle applicazioni di utilizzare le transazioni XA "a regime di controllo stretto ("tightly-coupled")" con XID differenti ma con stesso GTRID, è necessario impostare SSTRANSTIGHTLYCPLD sul parametro flags del metodo XAResource.start. Per ulteriori informazioni su come utilizzare questo flag, vedere Informazioni sulle transazioni XA.
