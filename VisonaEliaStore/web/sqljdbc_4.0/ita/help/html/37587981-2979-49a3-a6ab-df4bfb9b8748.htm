Metodi

Nome

Descrizione

getCatalogName

Ottiene il nome del catalogo per la tabella che include la colonna designata.

getColumnClassName

Restituisce il nome completo della classe Java di cui vengono prodotte le istanze se viene chiamato il metodo getObject della classe SQLServerResultSet per recuperare un valore dalla colonna.

getColumnCount

Restituisce il numero di colonne nel set di risultati.

getColumnDisplaySize

Restituisce la larghezza massima normale della colonna designata espressa in caratteri.

getColumnLabel

Ottiene il titolo suggerito da utilizzare in stampe e visualizzazioni della colonna designata.

getColumnName

Ottiene il nome della colonna designata.

getColumnType

Recupera il tipo SQL della colonna designata.

getColumnTypeName

Recupera il nome del tipo specifico del database della colonna designata.

getPrecision

Ottiene il numero di cifre decimali per la colonna designata.

getScale

Ottiene il numero di cifre a destra del separatore decimale per la colonna designata.

getSchemaName

Ottiene il nome dello schema di tabella per la colonna designata.

getTableName

Ottiene il nome di tabella della colonna designata.

isAutoIncrement

Indica se la colonna designata viene numerata automaticamente e, pertanto, resa di sola lettura.

isCaseSensitive

Indica se per una colonna viene fatta distinzione tra maiuscole e minuscole.

isCurrency

Indica se la colonna designata è un valore di cassa.

isDefinitelyWritable

Indica se un'operazione di scrittura nella colonna designata riuscirà sicuramente.

isNullable

Indica il supporto dei valori Null nella colonna designata.

isReadOnly

Indica se la colonna designata non è accessibile in scrittura.

isSearchable

Indica se la colonna designata può essere utilizzata in una clausola WHERE SQL.

isSigned

Indica se i valori della colonna designata sono numeri con firma.

isSparseColumnSet

Indica se una colonna in un set di risultati è un set di colonne di tipo sparse.

isWritable

Indica se è possibile che un'operazione di scrittura nella colonna designata venga completata.
