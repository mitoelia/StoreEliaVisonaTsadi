Metodi

Nome

Descrizione

free

Libera l'oggetto NCLOB e rilascia le risorse bloccate.

getAsciiStream

Recupera il valore NCLOB designato dall'oggetto java.sql.NClob come flusso ASCII.

getCharacterStream

Recupera il valore NCLOB designato dall'oggetto java.sql.NClob.

getSubString

Recupera una copia della sottostringa specificata nel valore NCLOB designato dall'oggetto java.sql.NClob.

length

Recupera un numero di caratteri nel valore NCLOB designato dall'oggetto java.sql.NClob.

position

Recupera la posizione dei caratteri dell'oggetto java.sql.NClob specificato o della sottostringa in java.sql.NClob in base alla posizione di inizio specificata.

setAsciiStream

Recupera un flusso da utilizzare per scrivere caratteri ASCII nel valore NCLOB rappresentato da questo oggetto java.sql.NClob, a partire dalla posizione specificata.

setCharacterStream

Recupera un flusso da utilizzare per scrivere un flusso di caratteri Unicode nel valore NCLOB rappresentato da questo oggetto java.sql.NClob, a partire dalla posizione specificata.

setString

Scrive l'oggetto String specificato in NCLOB, a partire dalla posizione specificata.

truncate

Tronca il valore NCLOB in base alla lunghezza specificata.
