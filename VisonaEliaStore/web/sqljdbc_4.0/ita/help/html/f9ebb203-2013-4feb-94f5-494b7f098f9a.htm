Nome

Descrizione

getParameterClassName

Recupera il nome completo della classe Java le cui istanze devono essere passate al metodo setObject della classe SQLServerPreparedStatement.

getParameterCount

Recupera il numero di parametri nell'oggetto SQLServerPreparedStatement per il cui l'oggetto SQLServerParameterMetaData contiene informazioni.

getParameterMode

Recupera la modalità del parametro designato.

getParameterType

Recupera il tipo SQL del parametro designato.

getParameterTypeName

Recupera il nome del tipo specifico del database del parametro designato.

getPrecision

Recupera il numero di cifre decimali per il parametro designato.

getScale

Recupera il numero di cifre a destra del separatore decimale per il parametro designato.

isNullable

Recupera un valore che indica se nel parametro designato sono consentiti valori Null.

isSigned

Recupera un valore che indica se i valori per il parametro designato possono essere numeri con segno.
