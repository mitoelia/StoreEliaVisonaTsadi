Metodo setDateTimeOffset (SQLServerCallableStatement)

Questo metodo è stato aggiunto in Microsoft JDBC Driver 3.0 per SQL Server.

Imposta il valore della colonna specificata sul valore Classe DateTimeOffset.

Sintassi

public void setDateTimeOffset(String sCol, microsoft.sql.DateTimeOffset t)

Parametri

sCol

Nome di una colonna.

t

Oggetto Classe DateTimeOffset.

Eccezioni

SQLServerException

Osservazioni

È possibile recuperare un valore Classe DateTimeOffset con SQLServerCallableStatement.getDateTimeOffset.

setDateTimeOffset accetta il numero ordinale della colonna.
