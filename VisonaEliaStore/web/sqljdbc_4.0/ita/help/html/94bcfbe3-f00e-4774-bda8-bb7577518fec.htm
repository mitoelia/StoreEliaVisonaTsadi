Connessione a SQL Server con il driver JDBC

Una delle operazioni più importanti che è possibile eseguire con Microsoft JDBC Driver per SQL Server è la creazione di una connessione a un database di SQL Server. Ogni interazione con il database viene eseguita mediante l'oggetto SQLServerConnection e poiché il driver JDBC ha un'architettura flat, quasi tutte le operazioni determinanti interessano l'oggetto SQLServerConnection.

Se SQL Server è in attesa solo su una porta IPv6, impostare la proprietà di sistema java.net.preferIPv6Addresses per assicurarsi che venga utilizzata la porta IPv6 anziché IPv4 per connettersi a SQL Server:

System.setProperty("java.net.preferIPv6Addresses", "true");

Negli argomenti di questa sezione viene descritto come creare e utilizzare una connessione in un database di SQL Server.
