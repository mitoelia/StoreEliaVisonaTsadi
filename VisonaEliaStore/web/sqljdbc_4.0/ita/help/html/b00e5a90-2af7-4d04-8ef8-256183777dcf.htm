Classe SQLServerConnectionPoolDataSource

Rappresenta connessioni di database fisiche per le gestioni dei pool di connessioni.

Pacchetto: com.microsoft.sqlserver.jdbc

Estende:SQLServerDataSource

Implementa: javax.sql.ConnectionPoolDataSource

Sintassi

public class SQLServerConnectionPoolDataSource

Osservazioni

SQLServerConnectionPoolDataSource viene in genere utilizzato negli ambienti dei server applicazioni Java che supportano il pool di connessioni predefinito e richiedono un oggetto ConnectionPoolDataSource per fornire connessioni fisiche, quali i server applicazioni Java EE (Java Platform, Enterprise Edition) che forniscono il pool di connessioni della specifica dell'API di JDBC 3.0.
