Gestione dei metadati con il driver JDBC

È possibile utilizzare Microsoft JDBC Driver per SQL Server per gestire metadati in un database di SQL Server in molti modi diversi. Il driver JDBC può essere utilizzato per estrarre i metadati del database, un set di risultati o parametri.

Il driver JDBC fornisce inoltre le tre classi seguenti per il recupero dei metadati da un database SQL Server:

  • SQLServerDatabaseMetaData, utilizzata per restituire le informazioni sul database con connessione attiva.

  • SQLServerResultSetMetaData, utilizzata per restituire le informazioni sul set di risultati.

  • SQLServerParameterMetaData, utilizzata per restituire le informazioni sui parametri di istruzioni preparate e richiamabili.

Negli argomenti di questa sezione viene illustrato come utilizzare ciascuna delle tre classi per operare con i metadati in un database SQL Server.

I metodi per metadati illustrati in questa sezione sono in genere dispendiosi in termini di prestazioni dell'applicazione e si consiglia pertanto di utilizzarli con molta attenzione.
