updateDateTimeOffset (SQLServerResultSet)

Questo metodo è stato aggiunto in Microsoft JDBC Driver 3.0 per SQL Server.

Aggiorna il valore della colonna specificata a un valore Classe DateTimeOffset.

Elenco degli overload

Nome

Descrizione

updateDateTimeOffset(int, microsoft.sql.DateTimeOffset)

Aggiorna il valore della colonna specificata al valore Classe DateTimeOffset, secondo il numero ordinale di colonna in base zero.

updateDateTimeOffset(string, microsoft.sql.DateTimeOffset)

Aggiorna il valore della colonna specificata al valore Classe DateTimeOffset, in base al nome di una colonna.
