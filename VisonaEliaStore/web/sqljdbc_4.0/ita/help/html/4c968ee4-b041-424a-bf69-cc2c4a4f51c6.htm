Metodo setIntegratedSecurity (SQLServerDataSource)

Imposta un valore Boolean che indica se la proprietà integratedSecurity è abilitata.

Sintassi

public void setIntegratedSecurity(boolean enable)

Parametri

enable

true se integratedSecurity è abilitata. In caso contrario, false.

Osservazioni

Impostare la proprietà su "true" per indicare che in SQL Server verranno utilizzate le credenziali di Windows per autenticare l'utente dell'applicazione. Se impostata su "true", tramite Microsoft JDBC Driver per SQL Server vengono cercate nella cache delle credenziali del computer locale le credenziali già specificate per l'accesso al computer o alla rete. Se impostata su "false", è necessario fornire il nome utente e la password.

Questa proprietà è supportata solo nei sistemi operativi Microsoft Windows.

Per ulteriori informazioni sull'utilizzo dell'autenticazione integrata, vedere Costruzione dell'URL della connessione.
